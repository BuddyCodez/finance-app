# 🚀 FundTracker

**Project Name**: FundTracker  
**Technologies**: Next.js, Shadcn UI, Clerk Authentication, Drizzle ORM, NEON DB  

## ✨ Current Features

- 📊 **Analytics of Transactions**: Get detailed insights into your financial transactions.
- 📝 **Manage Transactions**: Users can create, delete, and edit transactions.
- 🏦 **Manage Accounts**: Users can create, delete, and edit accounts (not for login/signup, used like bank accounts).
- 📂 **Categories**: Create and manage transaction categories.
- 🗂️ **Categorize Transactions**: Users can categorize their transactions.

## 🌟 Special Features

- 📥 **Bulk Create Transactions**: Upload a CSV to create multiple transactions at once.
- 🗑️ **Bulk Delete**: Remove multiple transactions, categories, or accounts in one go.

### Installation

To get started with FundTracker, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/fundtracker.git
    ```

2. Install dependencies:
    ```bash
    cd fundtracker
    npm install
    ```

3. Set up environment variables:
    - Create a `.env.local` file and add necessary environment variables.

4. Run the development server:
    ```bash
    npm run dev
    ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
